## 開発者向け情報

### ローカル開発環境のセットアップ

1. リポジトリをクローン
2. ローカルサーバーを起動
3. ブラウザでアクセス

```bash
# 簡単なHTTPサーバーを起動
python -m http.server 8000
# または
npx serve .
```

### カスタマイズ

- **色の変更**: `assets/css/style.css`のCSS変数を編集
- **コンテンツの変更**: `index.html`を編集
- **機能の追加**: `assets/js/script.js`を編集
